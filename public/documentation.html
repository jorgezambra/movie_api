Documentation

Mah Movies is a web application that will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

The table below describes each endpoint of this API and how to access it.

Description URL HTTP method Request body data format Response body data format
Get a list of all movies /movies GET None JSON object holding data about all movies. Structure:

{ Title: "The Godfather", Description: "Crime, Mafia movie...", Genre: {Name: "Crime", Description: "Movie involving criminal activity"}, Director: {Name: "Francis Ford Coppola", ImagePath: "casablanca.png", Featured: false}

Get data about a single movie by title /movies/[title] GET None JSON object holding data about a single movie. Contains same structure as previous example (GET /movies)
Get data about a movie genre /genre/[name] GET None JSON object holding data about a movie genre, containing name and description. Structure:

{Name: "Action", Description: "Movie is fast paced usually violent, etc."}

Get data about a director /director/[name] GET None JSON object holding data about a director. Structure:

{Name: "Francis Ford Coppola", Bio: "American Director, etc.", birthday: "04-07-1939", Death: ""}

Add a new user to the registry. /users POST JSON object holding data about the user to add. Structure:

{ username: "Jake", password: "aka3", email: "jake.wake@gmail.com", dateofbirth: "12-12-2001"

JSON object holding data about the user that was add. Structure:

{ id: 7, username: "Jake", password: "aka3", email: "jake.wake@gmail.com", dateofbirth: "12-12-2001"

Update a user's information /users/[username] PUT JSON object holding data about the user to add. Structure:

{ username: "Jake", password: "aka3", email: "jake.wake@gmail.com", dateofbirth: "12-12-2001"

A message indicating the user's information was updated.
Add a movie to the user's favorites list /users/[username]/favorites/[moviename] POST None A message indicating the movie title was added to the user's favorites list.
Delete a movie from the user's favorites list /users/[username]/favorites/[moviename] DELETE None A message indicating the movie title was deleted from the user's favorites list.
Remove a user from the registry. /users/[username] DELETE None A message indicating the user was removed from the registry.
